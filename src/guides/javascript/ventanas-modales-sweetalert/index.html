Crear una alerta simple

Mostrar una alerta simple, podría ser el 'hola mundo' de la librería

Evento de alerta cerrada o aceptada

Si queremos saber cuándo se cierra una alerta (es decir, que se presiona el botón de OK o que se cierra) espera a que se resuelva la promesa que el método .fire() devuelve

Swal.fire("Esta es una alerta")
    .then(() => {
        // aquí la alerta se ha cerrado
        console.log("Alerta cerrada");
    });

Podemos ver el resultado presionando el siguiente botón: y luego revisar la consola del navegador con CTRL + SHIFT + I (google chrome).

Cambiar texto del botón de alerta con Sweet Alert 2

Si no queremos que aparezca el texto 'OK' en el botón, podemos invocar a Swal.fire pero con un objeto de propiedades. Solo debemos especificar las propiedades title y confirmButtonText

Swal.fire({
    title: "Mensaje enviado",
    text: "El mensaje fue enviado exitosamente",
    confirmButtonText: "Aceptar"
})

Crear una alerta que contenga una imagen personalizada

Para añadir una imagen debemos establecer la url del recurso en la propiedad imageUrl dentro del objeto al invocar el método fire()

Swal.fire({
    title: "Mensaje enviado",
    text: "El mensaje fue enviado exitosamente",
    imageUrl: "https://robohash.org/2"
})
